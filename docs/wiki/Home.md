# MetaUI Documentation Wiki

Welcome to the official technical documentation for MetaUI. 

MetaUI is a plugin-driven JSON-schema rendering engine for SAP UI5. It dynamically generates Fiori layouts at runtime, removing the need for manual XML View construction. 

All code is strictly typed TypeScript and follows a zero-hack, highly modular architecture.
